# oss_2 project
### VimGolf 
---

**[설명]**
- 특정 문제를 최소한의 타수로 편집하는 사이트
- 타이핑하는 Keystroke를 최소한으로 맞춰라!
- <http://vimgolf.com>


**[설치방법]**
- apt update
- apt install python
- apt install python3-pip
- pip intsall vimgolf


**[문제 리스트 표시]**
- vimgolf list
- 하지만, 사이트들어가면 더 많은 문제 리스트 볼 수 있다.


**[특정 문제 참가방법]**
- vimgolf put [problem hashID]
